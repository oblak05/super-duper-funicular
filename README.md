# Instagram Follower Bot

This tool sends followers to an Instagram account using various Turkish follower services.

## Installation

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

### As a Python Tool

Run the script directly:
```bash
python v2.py
```

Follow the prompts to enter your fake Instagram account credentials and the target username.

### As a Web Application

Run the Flask web app:
```bash
python app.py
```

Open your browser to `http://127.0.0.1:5000/` and fill in the form to start the bot.

Use at your own risk. This may violate Instagram's terms of service. Use fake accounts as recommended.
